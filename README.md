# Student Management System

A Java console-based application for managing student records using Object-Oriented Programming principles.

## Features

* Add Student
* View Students
* Search Student
* Update Student
* Delete Student
* Duplicate ID Validation
* Input Validation
* Exception Handling
* Persistent Storage using students.txt

## Technologies Used

* Java
* OOP
* Collections Framework (ArrayList)
* File Handling
* Exception Handling

## Concepts Demonstrated

* Classes and Objects
* Encapsulation
* CRUD Operations
* Data Persistence
* Input Validation

## Project Structure

StudentManagementSystem

├── Main.java

├── Student.java

├── StudentManager.java

└── students.txt
